# ⚡ JNYANANJAN // Link Interface

> A futuristic, Links Hub designed as a **system interface**, not a website. **Made with AI by Me.**

---

## 🧠 Concept

This project is not a traditional portfolio.

It is built as a **Neural OS-style interface** where:
- Each section behaves like a **system module**
- Navigation feels like **accessing a machine**
- UI is inspired by **glassmorphism + cyberpunk minimalism**

> The goal: make the user feel like they are *interacting with a system*, not scrolling a page.

## 📱 Features

- ⚡ **Mobile-first design** (primary focus)
- 🧊 Glassmorphism UI with blur effects
- 🎯 Thumb-friendly navigation
- 🧠 System-style labels (ACTIVE / SYNCING / IDLE)
- 🎛️ Minimal yet high-impact layout
- 📡 External links to real platforms (GitHub, Startup, Socials)
- 🔊 Tap vibration feedback (on supported devices)

## 🧬 Customization

You can easily customize:

- Name → inside `index.html`
- Links → update `<a href="">`
- Colors → inside Tailwind config
- Layout → tweak `.card` in CSS
